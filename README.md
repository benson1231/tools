# tools
Utility Functions and Tools for Bioinformatics Analysis

## 創建環境
```bash
# create conda env
mamba env create -f envs/bio-tools.yaml -y
mamba activate bio-tools
```

## 離開環境與移除環境

```bash
# exit conda environment
mamba deactivate
# remove conda environment
mamba env remove -n bio-tools -y
````
